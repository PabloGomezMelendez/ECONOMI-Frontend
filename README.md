# 🎨 ECONOMI-Frontend

Este repositorio contiene el código fuente del frontend del proyecto ECONOMI, una plataforma orientada a la gestión de recursos económicos y financieros. Su objetivo es ofrecer una interfaz moderna, intuitiva y responsiva que permita a los usuarios interactuar con los servicios del backend de forma eficiente.

## 📁 Estructura del proyecto

- `src/components/` – Componentes reutilizables
- `src/pages/` – Vistas principales
- `src/services/` – Conexión con la API
- `src/store/` – Gestión de estado
- `public/` – Archivos estáticos
